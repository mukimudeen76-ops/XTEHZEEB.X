02 — Architecture
Built to update itself.
GH
GitHub Releases
Every build is tagged and published as a release, source of truth for what "latest" means.
λ
Serverless proxy
A Vercel function holds the access token server-side and asks GitHub for the newest release on demand. The token never reaches the browser.
UI
This page
Fetches from that function on load, so the download button always points at whatever shipped most recently — no redeploy needed.
